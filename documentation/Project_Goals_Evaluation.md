# Project Goals Evaluation

| Goal | Status | Evaluation |
|---|---|---|
| Upload unofficial transcripts | Met | PDF and DOCX extraction are supported. |
| Detect completed courses | Met | Passing courses are identified and normalized. |
| Exclude failed or withdrawn courses | Met | Failing and withdrawal grades are filtered. |
| Exclude courses in progress | Met | Parsing stops before the in-progress section. |
| Exclude co-op courses | Met | COOP subjects are explicitly excluded. |
| Track degree progress | Met | Completed courses are compared with program requirements. |
| Support course selection | Met | Students can review and select eligible courses. |
| Detect schedule conflicts | Met | Selected sections can be checked for overlap. |
| Generate a curriculum plan | Met | AI-assisted semester recommendations are supported. |
| Provide production deployment | Partially met | The application works locally; final cloud deployment may need additional configuration. |
| Handle every transcript layout | Partially met | Common Wentworth layouts work, but unusual PDF spacing remains an edge case. |
