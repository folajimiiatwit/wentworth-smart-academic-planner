# User Manual

## 1. Start the Application
Start the backend service and then launch the Streamlit frontend.
Also accessible through this [link](https://wentworth-smart-academic-planner.onrender.com/).

## 2. Log In
Enter the required student information on the login page.

## 3. Upload a Transcript
Upload an unofficial transcript in PDF or DOCX format. The system extracts completed courses and ignores withdrawn, failed, excluded co-op, and in-progress courses.

## 4. Review Completed Courses
Confirm the detected completed courses. Manually correct any course that was not detected because of an unusual transcript format.

## 5. Review Degree Progress
The application compares completed courses with the Computer Science degree requirements and shows completed and remaining requirements.

## 6. Select Courses
Choose from eligible courses. The system uses prerequisite and completion information to prevent inappropriate selections.

## 7. Build a Schedule
Add course sections to the schedule. The planner identifies overlapping meeting times and helps create a conflict-free schedule.

## 8. Generate a Curriculum Map
Use the curriculum-planning feature to generate a recommended sequence of future semesters based on remaining requirements.

## 9. Save or Export
Use the available application controls to preserve the selected plan or present it to an academic advisor.

## Known User Limitation
Transcript PDFs vary in layout. Students should verify imported courses before relying on the plan.
