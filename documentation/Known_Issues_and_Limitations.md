# Known Issues and Limitations

1. PDF transcript text extraction may merge words and grades, such as `PROGRAMMINGLANGUAGESB+`.
2. Transcript layouts from other institutions may require new parsing rules.
3. Users should verify imported courses before making registration decisions.
4. AI recommendations depend on API availability, configuration, and prompt quality.
5. AI-generated plans are advisory and should be reviewed against official catalog and advisor guidance.
6. Authentication is simplified and is not intended to replace institutional single sign-on.
7. File-based data storage is appropriate for the prototype but not ideal for a large production deployment.
8. Deployment configuration, secrets management, logging, and monitoring require additional production hardening.
